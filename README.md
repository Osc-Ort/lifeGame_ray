*lifeGame_ray*
Simple simulador del John Conway's Game of Life hecho en Python con raylib.

Espacio: parar la simulación.
Cuando parado ->
|> E: Limpieza de todo el tablero.
|> Click izquierdo: Rellenar celda.
|> Click derecho: Borrar celda

Q: Reinicio aletorio

Para ejecutar con uv, clonar el repositorio y ejecutar:
```bash
    uv sync && uv run main.py
```
